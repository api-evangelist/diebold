# Diebold Nixdorf (diebold)

Diebold Nixdorf is a global leader in driving connected commerce for the financial and retail industries, providing self-service kiosks, ATMs, point-of-sale systems, the Vynamic software platform, and related services. While Diebold Nixdorf operates customer-facing support and software portals such as the Aftersales Portal, eServices Portal, and Manuals/Drivers/Downloads portal, it does not currently publish a public open developer portal or open APIs. This repository captures the APIs, developer tools, and machine-readable API artifacts for Diebold Nixdorf as they become available.
